# 宫本武藏
```
2020-02-29
```
```
流浪的剑客，修行的武者。
顽石未启，恍惚踉跄徘徊。
乱花迷眼，痴痴几落深渊。
雨落巨杉，荡涤心中尘垢。
寺中听禅，惊醒迷途知返。
战斗，读书，习剑，登山。
书法，雕刻，谈心，垦田。
寻强求一败，博取诸家长。
沉思于故事，古今看兴衰。
外练筋骨强，内修心沉淀。
修行无定法，天地一形骸。
攀登雪山，开垦荒原。
激流清冽，雪落天寒。
诚心求道，岁岁年年。
登堂入室，心有质变。
妖魔挡路，山高风寒，勇士岂惧路艰险。
情之至深，泪洒花间，豪杰亦会心太软。
清流入海，尘世繁华如云烟，穿过人群喧闹处，海阔天蓝。
一人独面，刀剑霍霍耀人眼，虎破夜暗啸深渊，剑法自然。
修行岂有极，四季无暑寒。
纵浪大化中，翩然天地间。
```
