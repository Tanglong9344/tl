# 人为什么活着
##### 2019//11/12
```
人为什么活着？
似乎是为了功名利禄、吃穿享用、功成名就......甚至是文明发展、社会进步。
而所有的这些，最终又有什么意义呢？
一头猪整日吃吃睡睡，顶多在无聊的时候围着猪圈里转几圈，这一生又有什么意义呢？
是为了长到膘肥体壮的时候被杀了奉献浑身的肉吗？
这仅仅是从人的角度来看，尤其是喜欢红烧肉的人。
然而人并不是世界的中心，只是无数生命体中普普通通的一部分。
从呱呱坠地到重归泥土之间的过程，或者在其他生命看来和猪羊的一生一样也没什么太大区别。
所谓文明也只不过是人类画了一个圈，然后在圈子里自娱自乐，陶醉在自己的文明社会里，忘乎所以。
宇宙深处隐隐约约传来一个声音：人生本没有任何意义，所有的生命都只不过是一个存在罢了。
```
