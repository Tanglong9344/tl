# 人为什么活着
```
 2019//11/12
```
---
```
人为什么活着？
似乎是为了功名富贵、吃穿享受，
为了虚荣心、成就感。
为了追逐利益、贪图享受。
```
---
```
人活着不仅仅是为了自己，
还有理想道义，还有责任担当，还有所爱之人。
这么看来人活着有一部分是为了情感。
人要学会分工合作，相互成就，互帮互助，才能共渡难关，
才能创新科技、进步社会、发展文明。
贪婪自私的无限索取换来的到最后只有悲凉和毁灭。
这么看来，人活着有一部分是为了群体的发展。
孤独的个体寻求群体的温暖，
或许仅仅只是残酷环境下生存的需要，
而情感则是最好联系纽带。
即使有文明这个遮羞布，也无法完全掩盖趋利避害的本能。
```
---
```
一头猪整日吃吃睡睡，
顶多在无聊的时候围着猪圈里转几圈，
这一生又有什么意义呢？
是为了长到膘肥体壮的时候被杀了奉献浑身的肉吗？
这仅仅是从人的角度来看，尤其是喜欢红烧肉的人。
然而人并不是世界的中心，只是无数生命体中普普通通的一部分。
从呱呱坠地到重归泥土之间的过程，
或许在其他生命看来和猪羊的一生一样也没什么太大区别。
所谓文明也只不过是人类画了一个圈，
然后在圈子里自娱自乐，陶醉在自己的文明社会里，忘乎所以。
```
---
```
文明总会有边界，发展总会有尽头。
人类有生死，宇宙有存灭，所有的一切都会重归于虚无。
而所有的这些，最终又有什么意义呢？
```
---
```
宇宙深处隐隐约约传来一个声音：
人生本没有任何意义，所有的生命都只不过是一个存在罢了。
存在着，
就像天空中东升西落、氢核聚变的太阳；
像水底游来游去的鱼群，像随风摇曳的花草；
像巍然耸立的高山，像奔流不息的江河；
像万物之间相互作用的万有引力，像飞舞的埃尘，像爱因斯坦的沉思；
像眼底的泪水、嘴角的笑意、睡梦中呢喃；
像众生、像万物、像富贵，像生死。
```
