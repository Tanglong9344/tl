# 我所不理解的怪现象
```
2020-03-06
```
```
不管是离校之前还是之后，我常常会遇到一种怪现象。
当有的人有求于别人或觉得别人比他强出很多的时候，
就会变得格外的卑贱，
会把称呼由直呼姓名改成某某大佬、某某大神，甚至是某哥、某爷，
他们会忽然主动跟你套近乎，辛勤地帮你做事，朋友圈会积极点赞，
态度极其谦恭，笑容极其可亲，就是要令你无法拒绝。
```
```
如果你善意地帮助了他们，他们也并不会心存感激，
只是理所当然地认为这是他们应得的。
在他们眼里，你只不过是被他们利用的工具，
他们把伪装和欺骗当做竞争技能，并引以为傲。
而当你陷入困境的时候，他们只会冷眼旁观，
甚至落井下石，踩上几脚。
这些人不是智商有问题，
就是严重缺乏自信，或者二者都有。
在这里不讲道德，因为道德充满了不确定性，太过依赖群体。
这种摇尾乞怜的方式多数时候解决不了任何问题，
换来的只是他人蔑视与嘲弄的眼神。
只有能够解决各种实际问题的硬实力，
才能获得更多的资源和机会，
而不是逢人就扬起的笑脸或者卑微到尘土里的态度，
更不是多层的伪装或者各样的欺骗。
```
```
他们应该也明白这个道理，
只不过他们没有把心思放在如何让自己强大起来上，
而是整日想着投机钻营，如何借势升天。
他们一味索取，却很少付出。
他们以自己为世界的中心，
像苍蝇寻找鲜肉一样，嗡嗡地搜寻自己的猎物。
```
```
当然，他们还是会遇到一些单纯的善良人，
愿意帮助他们，而这些善良人也只不过是他们宰杀的又一批羔羊。
慢慢地，这些羔羊要么继续善良，任人宰割，
要么丢掉善良，变成嗜血凶狼，
或者只把善良留给善良的人，面对恶狼则亮起手中的刀枪。
与人初识，待之诚善，若得诚善，则继续诚善；
若反得欺慢，则加倍还之，亦不复诚善。
既无血缘关系，又无利益关系，
也没有什么感情可谈，谁也没必要惯着谁。
```
```
正常的合作方式，应该是互帮互助，各取所需，互利共赢，
而不是某一方一味索取，却极少付出。
以德报怨的事，我不会做。
不要做没有脾气的笑脸老好人，
而是做一个普通的正常人，有所不为，有所必为。
不愿与他们同流合污，又要保护自己，就只能敬而远之，
避无可避就只好拿起武器，以牙还牙。
```
```
如果这个世界上到处流窜的都是贪婪的饿狼，
这些饿狼又该以何为食呢？要么自相残杀，要么死在猎人的枪下。
这个社会既不需要羊，也不需要狼，
没有狼，也就不需要刀枪。
需要的只是能够正常生活的烟火人。
```
```
或许这只是一个遥不可达的梦幻。
```
