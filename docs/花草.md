### 花草
```
2020-02-22
```
```
经过漫长寒冬的挣扎，
草儿遇到花儿。
天真地以为可以一起
走过寒暑，
共看朝阳、晚霞。

却完全没听到，
梦幻破裂的喑哑，
像极了天空刺眼的烟花。

所有的美好不过一个童话，
故事讲完，
就结束了。

笑着笑着就哭了，
哭得像个傻瓜。
海誓山盟不过一个测试，
而你没有正确作答。

你不过是她随意的规划，
和谁不能成家？
平凡的生活本就无需太多，
所谓爱情不过是个笑话。

你不会懂，
因为花草本就是
两个不同的物种。
几个蹩脚的借口，
就可以转身离去。
一阵微风，
即可抚平伤痛。

怎能妄想获得玫瑰的芳心。
你不过是一棵野草，
不配她的信任。
好像做梦的牛粪。

野草只能被火烧，
你的天真，
只被嘲笑。
花儿有自己的美好。
而草儿只是肥料。
```
