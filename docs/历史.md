# 历史
### 2019/09/22
```
过去的事谁又说得清楚到底哪些是真的，哪些是假的呢？
我们现在所知道的，
要么是第三者的记录，
要么是一些人并不一定准确的推理甚至仅仅只是想象而已。
绝大部分事实都不会被完全展现甚至会完全相反，
因为这些东西都是人记录下来的，
记录者会有自己的目的、意志和情感。
我们并不知道当初到底发生了什么事情，
我们只能看到现在存在着的结果。
所谓的历史唯物主义也只能进行局部的修正和调整，
现在永远无法完全得到过去的真相。
而且过去的事实好像也并不怎么重要，
人们只想看到他们所希望看到的真相，
因为大多数人只在乎和他们相关的结果。
```
---
```
尽管如此，历史的依然意义重大。
漫长的历史并不是人们满足虚荣心的炫耀资本，
ta有着更重大的作用。
司马光注解资治通鉴为：鉴于往事，以有资于后世。
历史在一定程度上会帮我们驱散部分疑惑，
让我们在陷入困境时不至于完全仓皇失措。
历史带给我们的不仅仅是沧桑的厚重感，
也不仅仅是满足人们猎奇探究的心理，
漫漫黄沙之中还有很多有用的经验教训，
有效的方式方法，睿智明达的见解思想。
我们虽然因为种种原因无法完全重现过去的事实，
但如果我们能够结合历史的经验教训，
具体问题具体分析，
就可以更快地找到更有效的解决方案。
```
---
```
历史的范围何其广泛。
ta包含的不只是几千、几百、几十年前发生的事情，
ta包含过去几年、几天甚至几小时、几分钟的存在。
ta囊括的不仅是别国、别地、别人过去的一切，
同样覆盖每个人自己的生活和经历。
ta不仅仅是生命体的物质、信息和能量
还有所有非生物的运动和变化。
ta横跨时间的无始无终和空间的无穷无尽，
从原子的概率运动和能量跃迁
到宇宙的大爆炸和伸缩演变。
现在和未来最终都会成为历史的一部分。
在历史的大数据面前，人的一生渺若埃尘。
```
---
```
人和其它生物和非生物一样，
从哪里来？到哪里去？为什么而存在？
现在没有答案，以后也不可能找到真正的答案。
所有的存在都仅仅只是一个存在。
```
