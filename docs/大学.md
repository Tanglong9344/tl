# 大学
##### 2019/11/12
```
读大学的目的仅仅是系统地学习一套专业知识吗？
学校里学到的那些知识一部分以后永远也不会再用，
剩下的大部分在这个信息更迭迅速的时代也会很快过时。
更大的学习平台实际上是现实的社会，是我们这个世界，
而最好的老师则是我们日常的生活。
大学更重要的思维方式的培养，
是各种学习、探索和研究的方式方法的积累及挖掘，
是意志的锻炼和习惯的养成。
通过大学我们可以对这个城市的文化、历史、经济、
风俗习惯、地理名胜等进行充分的了解，最终收获的是一个城市。
了解一个城市的途径有很多，旅游，生活、工作等。
而读大学是最有效的，大学有很多同学可以交流，
有很多老师可以引导，有很多资源可以使用，有很多时间可以自由安排。
大学学的可不仅仅是专业知识这么简单。
```
