# 为什么要读大学
```
2019/11/12
```
---
```
读大学的目的仅仅是系统地学习一套专业知识
并最终获得一份文凭吗？
学校里学到的那些知识
一部分以后永远也不会再用，
剩下的大部分在这个信息更迭迅速的时代
也会很快过时。
文凭既有用也无用，
它可以作为一把钥匙
帮我们找到一份理想的工作，
然而文凭只是静态地代表过去，
而且并不一定准确，
更确切地说
是作为和其它拥有文凭的人作比较的时候
判断谁胜任工作的概率更高的依据之一。
相较之下，
更不可见的解决实际问题的能力和思维更重要。
不过在现实情况中，
文凭往往会在最初的时候占上风。
```
---
```
知识只代表过去，能力决定现在，
眼界、思维方式和学习力可以重塑未来。
大学只是这个世界一个较小的子集，
更大的学习平台实际上是现实的社会，
是我们这个世界，
而最好的老师则是我们日常的生活。
大学更重要的思维方式的培养，
是各种学习、探索和研究的
方式方法的积累及挖掘，
是意志的锻炼和习惯的养成。
```
---
```
通过大学我们可以对这个城市的文化、历史、
经济、风俗习惯、地理名胜等进行充分的了解。
了解一个城市的途径有很多，
旅游，生活、工作等，而读大学是最有效的。
大学有很多同学可以交流，有很多前辈可以请教，
有很多资源可以使用，有很多时间可以自由支配。
时间如果充足，我们最终会收获一个城市。
如果想要充分了解一个城市，就去这个城市读书吧。
```
---
```
一般情况下，
高学历和更高学历的人相比，最大的差别不是知识体系，
是观察问题的角度、思考问题的方式、解决问题的方案；
是对资源的统筹、信息的综合、自身的管理；
是学习和研究的能力，
是行动的效率和探索的精神，是心境和意志力；
是对人生的态度、对生活的理解、看世界的眼界。
```
---
```
高学历的人并不一定拥有这些能力，
即使有也不一定强。
只不过相对低学历和没学历的人来说，概率比较高。
低学历和没学历的人中有不少人也可能拥有这些能力，
可能还很强。
因为在社会和世界这个广阔的学习天地中，
Ta们独辟蹊径，历经风雨，
用实实在在的实践和不断地思考总结
一步步迈进生活的真谛，探得了世界的秘密。
Ta们是真正的勇者，是真正的智者，
Ta们真正拥有了自己的人生。
```
