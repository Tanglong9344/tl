# 《星际穿越》有感        
##### 2016/3/4
### 事事于心，
### 有万千感慨汹涌澎湃，
### 熊熊火焰，燃尽了往日云烟。
### 苦乐荣辱，虚假笑脸；
### 峥嵘岁月，纵马平川。
### 风雨之中，暗夜之下。
### 有沉思，有前瞻；
### 有星空，有奇点。
### 有万丈巨浪掩下千年的璀璨。
### 流星闪过，天际依然。
### E=mc2的宇宙相对了爱因斯坦。
### 走过的路途，脚印已被掩盖。
### 腾飞的巨龙，浴火的凤凰，乘风的大鹏。
### 西去的师徒，一路上妖魔鬼怪。
### 移山的愚昧，填海的无知，逐日的疲倦。
### 滚滚东去的江水拍打着沧桑的堤岸。
### 时间和空间的无垠连接了无尽的寻探。
